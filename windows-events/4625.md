# Event ID 4625 — Failed Login

## Description
This event indicates a failed login attempt in Windows.

---

## Why It Matters

Multiple failed logins may indicate:

- Brute force attacks
- Password spraying
- Misconfigured services
- Expired credentials

---

## Important Fields

| Field | Meaning |
|---|---|
| Account_Name | User account |
| Source_Network_Address | Source IP |
| Logon_Type | Login method |
| Workstation_Name | Origin workstation |

---

## Example SPL Query

```spl
index=wineventlog EventCode=4625
| stats count by Account_Name, host
```

---

## Investigation Steps

1. Identify the affected account
2. Verify source IP
3. Check repeated attempts
4. Look for successful login after failures
5. Investigate suspicious patterns
